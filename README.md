## Portfolio Website
- [jw7914.com](https://jw7914.com)

A modern, interactive portfolio website showcasing my projects and skills.
## Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Component Library**: [Aceternity UI](https://ui.aceternity.com)
  - **Animations**: [Framer Motion](https://www.framer.com/motion/)
  - **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/), [Tabler Icons](https://tabler-icons.io/), [Devicon](https://devicon.dev/)
- **Email Service**: [EmailJS](https://www.emailjs.com/)

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── final/        # Section-specific components (Skills, Projects, etc.)
│   └── ui/           # Generic UI elements (CometCard, Buttons, etc.)
├── lib/              # Utility functions (cn, etc.)
├── App.jsx           # Main application component
└── main.jsx          # Entry point
```
